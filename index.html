C ++ OpenGL Engine Demo

Completed May 2020

This is a custom C++ OpenGL rendering engine, i made in my final HND year as student at ESAT, Valencia.

Its main features are the following:

- ECS - Physically Based Rendering - PBS, PBR Workflow. - Image Base Lighting (Radiance and Irradiance environment mapping) - Postprocessing - Bloom - Shadow Mapping - Tone mapping - Gamma Correction - Color Inverter - Multithreaded agnostic graphics API - Shadow - Point light single pass. - ImGui integration. - Cross-platform (Windows and Linux)
